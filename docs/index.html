 Gráfica interactiva: Cada esfera representa un alumno. Usa tu mouse para girar, alejar, acercar y explorar la gráfica. Pasa el cursor por cada esfera para desplegar la información por alumno. El tamaño es proporcional a la CALIFICACIÓN promedio (0-100), la escala de color es proporcional al % de ASISTENCIA y, por último, la altura corresponde a tres categorías de CONDUCTA: 0 (MALA), 50 (REGULAR) y 100 (BUENA). Cada nivel de conducta indica el nivel de atención y comportamiento observado en el alumno durante las sesiones de aprendizaje.
